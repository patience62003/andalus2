# Andalus School — React/Vite (andalus2)
Deploy to Vercel:
1) Create a GitHub repo named **andalus2** and upload the *contents* of this folder.
2) Go to https://vercel.com → New Project → import **andalus2**.
3) Framework: **Vite**, Build: `npm run build`, Output: `dist`.
4) Deploy → URL like https://andalus2.vercel.app
PDFs go in `/public` and links use `/YourFile.pdf`.
